# Basic URL Phishing Detection App

## Project Description
This project detects whether a URL is phishing or legitimate using machine learning techniques.

## Features
- URL analysis
- Phishing detection
- User-friendly interface
- Fast prediction

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

## Installation
1. Clone repository
2. Install requirements
3. Run app.py
